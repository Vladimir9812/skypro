## Описание

Простой веб-сайт для ведения личных расходов.  
Позволяет добавлять, редактировать и удалять записи о тратах, а также просматривать их в виде таблицы.

## Деплой

Сайт доступен по ссылке: https://vladimir9812.github.io/skypro

## Функционал

- **Добавление** нового расхода (дата, категория, описание, сумма)
- **Редактирование** существующих записей
- **Удаление** расходов
- **Просмотр** всех расходов в таблице
